# Laporan Praktikum Pemrograman Web I

Repository ini merupakan dokumentasi resmi **tugas, latihan, dan laporan Praktikum Pemrograman Web I** yang disusun untuk memenuhi salah satu persyaratan pada mata kuliah Pemrograman Web I.

Seluruh isi repository ini dibuat berdasarkan materi yang diberikan selama proses perkuliahan dan praktikum.

---

## Tujuan Pembuatan Repository

Pembuatan repository ini bertujuan untuk:
1. Mendokumentasikan hasil praktikum Pemrograman Web I secara terstruktur.
2. Menyimpan arsip tugas dan laporan dalam bentuk digital.
3. Memudahkan proses evaluasi dan penilaian oleh dosen pengampu.
4. Menjadi referensi pembelajaran pemrograman web dasar.


---

## Materi Praktikum

Materi yang dipelajari dan diimplementasikan dalam praktikum ini antara lain:
- Pengenalan HTML dan struktur dasar halaman web
- Penggunaan elemen dan atribut HTML
- Penerapan CSS dasar untuk tampilan web
- Pembuatan layout halaman web sederhana
- Latihan dan studi kasus sesuai modul praktikum

---

## Cara Menjalankan Project

1. Clone repository ini ke komputer lokal:
   ```bash
   git clone https://github.com/fahrurozi911/PEMROGRAMAN-WEB-1.git

2. Masuk ke folder project.


3. Buka file .html menggunakan web browser.


4. Apabila terdapat file .php, jalankan menggunakan server lokal seperti XAMPP atau Laragon.
